Como funciona a conversão binária

A conversão de decimal para binário é feita dividindo o número por 2 repetidamente até o quociente ser 0.

Como funciona a conversão octal

A conversão de decimal para octal é realizada dividindo o número por 8, obtendo os restos.

Como funciona a conversão hexadecimal

A conversão de decimal para hexadecimal utiliza a base 16, representando números com dígitos de 0 a 9 e letras de A a F.
