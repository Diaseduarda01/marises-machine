Como funciona a conversão binária

A conversão de decimal para binário é feita dividindo o número por 2 repetidamente. Os restos obtidos, lidos de baixo para cima, formam o número binário.

Como funciona a conversão octal

A conversão de decimal para octal é realizada dividindo o número por 8. Os restos obtidos, lidos de baixo para cima, formam o número octal.

Como funciona a conversão hexadecimal

A conversão de decimal para hexadecimal usa a base 16. Os números são representados com dígitos de 0 a 9 e letras de A a F, com cada dígito representando um valor de 0 a 15.
