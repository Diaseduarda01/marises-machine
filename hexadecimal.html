Como funciona a conversão hexadecimal para binário

A conversão de hexadecimal para binário é feita convertendo cada dígito hexadecimal em seu equivalente binário de 4 bits.

Como funciona a conversão hexadecimal para decimal

Para converter hexadecimal para decimal, multiplica-se cada dígito do número hexadecimal pelo seu valor posicional (16^n) e soma-se os resultados. Por exemplo, o hexadecimal 2A é calculado como (2 × 16^1) + (10 × 16^0) = 32 + 10 = 42.

Como funciona a conversão hexadecimal para octal

Para converter hexadecimal para octal, primeiro converte-se o hexadecimal para decimal e, em seguida, o decimal para octal.
